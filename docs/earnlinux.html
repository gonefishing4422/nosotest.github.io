Earn Noso on Linux

To mine Noso Coin, you will need a wallet and a mining software. Step 1 Download the latest version of Nosolite wallet from Github.com (here) Wallet will generate a default address, you can generate as many addresses you need. Alternatively you can use Noso Web Wallet, download Noso Android wallet, or download Noso Wallet both found on Github (here). Step 2 (consominer) Mine on Linux Ubuntu using consominer (These steps install consominer v0.70. You should download the latest version located on Github (here). From within a terminal on your Linux machine, enter the following commands: sudo apt-get update -y && apt-get upgrade -y wget https://github.com/Noso-Project/consominer/releases/download/v0.70/consominer-v0.70-x86_64-linux.tar.gz tar -xvzf consominer-v0.70-x86_64-linux.tar.gz chmod +x consominer ./consominer Note, these steps assume x86 64bit hardware. If your miner uses arm or aarch64 CPU, you will need to use a different binary. Refer to “Mine Noso Coin with Android and Termux” for your ARM binary download. Step 3 After launching consominer, Select an address from your wallet and configure it by entering “address {YourNosoWalletAddress}” To determine how many CPUs (X) provide the highest hash rate for your hardware enter “test {CPU count}” After determining how many CPUs you want to use for mining, enter “cpu {X}” Optionally, to have your miner start automatically upon launch, enter “autostart true” Optionally, enter, “source clear” to solo mine or enter “source {IP:Port}” to pool mine. Optionally, if solo mining make sure to set your minerid. “minerid [0-8100]” Enter “mine” to start mining. Note: to pool mine, enter source pool:port. Enter source add pool:port to add additional “backup” pools. Refer to (pools) for the most updated pool list. Step 2 (noso-2m) Mine on Linux Ubuntu using noso-2m (These steps install noso-2m v0.2.2. You should download the latest version for your hardware located on Github (here). apt-get update -y && apt-get upgrade -y wget https://github.com/f04ever/noso-2m/releases/download/v0.2.2/noso-2m-v0.2.2-linux-x86_64.tar.gz tar -xvzf noso-2m-v0.2.2-linux-x86_64.tar.gz chmod +x noso-2m ./noso-2m -a {address} -t {threads} i {miner instance} –pools=Pool1:IPAddress:port;Pool2:IPAddress:port” MINING POOL EXAMPLE: ./noso-2m.exe -a ThisIsMyNosoMiningAddress -t 9 -i 6 –pools=Dev:45.146.252.103:8082;rukzuk:173.199.122.137:8082;f04ever:209.126.80.203:8082? MINING SOLO EXAMPLE: ./noso-2m.exe -a ThisIsMyNosoMiningAddress -t 9 -i 6 Note, these steps assume x86 64bit hardware. If your miner uses hardware other than x86 64, download your hardware appropriate version (here). Also, if you are SOLO mining with multiple instances, you must define your miner instance. HAPPY MINING!
